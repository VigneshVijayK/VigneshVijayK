CERTIFICATES GALLERY IMAGES:

Place your certificate images in this folder.

Suggested file names:
- ceh.jpg (CEH Certificate)
- eeh.jpg (EEH Certificate)
- copa.jpg (COPA Certificate)
- Add more certificates as needed

Certificate requirements:
- Format: A4 portrait orientation (210mm x 297mm or 8.27" x 11.69")
- Recommended resolution: 2480x3508 pixels (300 DPI) or higher
- Format: JPG or PNG
- File size: Optimize for web (under 1MB per certificate recommended)

The certificates will be displayed in A4 album format with lightbox functionality.
Users can click to view full-size certificates.
